# **App Name**: GainGraph

## Core Features:

- Live Profit Calculator: Real-time interactive inputs for investment amounts and multipliers, providing instant calculation of final amounts and profits.
- Precision Validation Engine: An automated system to ensure that all financial entries for amounts and multipliers are positive numbers before processing.
- Rupee-Centric Formatting: Locale-aware formatting of all currency fields and result displays using the Indian Rupee (₹) symbol.
- Strategic Growth Narrative: A generative AI tool that analyzes the calculated profit to provide context on the scale of the gain and its impact relative to common financial benchmarks.
- Dynamic Summary Dashboard: A high-fidelity display component showing entered data versus calculated outcomes for easy reading at a glance.

## Style Guidelines:

- The application uses a 'Royal Obsidian' theme based on the concept of wealth and digital precision. The Primary color is a sophisticated Royal Purple (#8E78D9).
- Background is a Deep Night Slate (#15121C), heavily desaturated and dark to enhance visual focus. The Accent color is a bright Periwinkle (#6088FF) to highlight interactive calculated states.
- A single font recommendation of 'Space Grotesk' (sans-serif) for both headlines and body text, emphasizing the technical and scientific nature of a financial calculator with its wide, computerized letterforms.
- A focused, single-page modular card layout with high-contrast text and significant whitespace, ensuring clear separation between input fields and calculated results.
- Fluid, buttery number scrolling for the final profit values to provide a sense of organic growth and modern responsiveness.
- Monochrome, line-weight thin icons used sparingly to designate input fields and profit summaries, avoiding visual clutter.
